Longest Substring Without Repeating Characters

问题描述

Given a string s, find the length of the longest substring without repeating characters.

Example 1:

Input: s = "abcabcbb"
Output: 3
Explanation: The answer is "abc", with the length of 3.

Example 2:

Input: s = "bbbbb"
Output: 1
Explanation: The answer is "b", with the length of 1.

Example 3:

Input: s = "pwwkew"
Output: 3
Explanation: The answer is "wke", with the length of 3.
Notice that the answer must be a substring, "pwke" is a subsequence and not a substring.

换而言之，就是给定一个字符串s，要求我们返回它的最长无重复连续的子字符串。

问题求解

这题第一下想的是把字符串改成数组，第一次出现就为0，并且把这个数标记，这样就可以把不同的字符变成相同的数字，再统计最长的连续0的长度即可。例如，s="abcabcbb"就可以变成nums=[0, 0, 0, 1, 1, 1, 1, 1]。但是这样有一个问题s="pwwkew"就会变成nums=[0, 0, 1, 0, 0, 1]，这么看来是2了，但是答案是3。究其原因是有些字符不是第一次出现的，它就是出现过了后面也是可以的，因此这种办法太粗糙了。下面是刚才说的错误解法。

解法一：暴力解法

这个方法没啥好说的，暴力就完事了。两层循环，第一层是子字符串的开头，第二层是子字符串的结尾。

解法二：滑动窗口算法

这个方法是我去问chatGPT，不管是GPT3.5还是GPT4.0，都要和我说的。我先自己大彻大悟一下。

在解决最长不重复子串问题时，我们可以使用可变大小的滑动窗口。我们维护一个窗口，该窗口在字符串中滑动，同时跟踪窗口中的字符。当遇到重复字符时，我们将窗口的左边界向右移动，直到没有重复字符为止。这样，我们就可以在遍历整个字符串的同时找到最长的不重复子串。
