Longest Palindromic Substring

问题描述

Given a string s, return the longest palindromic substring in s.

Example 1:

Input: s = "babad"
Output: "bab"
Explanation: "aba" is also a valid answer.

Example 2:

Input: s = "cbbd"
Output: "bb"

palindromic，也就是指回文。正着读和反着读都一样的就是回文。例如aba，aaghgaa等等。值得注意的是，如果全是一样的内容，也是回文。 题目要求我们找到一个字符串的最长的子字符串，且该字符串得是回文。那么判断回文就是一个很有意思的问题。这里给出两种判断方法。

将字符串倒叙输出，如果与原字符串相等，则为回文。

将字符串从中心处（奇数长度为中心，偶数长度则先判断中心的两个字符串应先相等）向两端延伸，两端的字符均相等则为回文。
