168. Excel Sheet Column Title

问题描述

Given an integer columnNumber, return its corresponding column title as it appears in an Excel sheet .

For example:

Example 1:

Input: columnNumber = 1
Output: "A"

Example 2:

Input: columnNumber = 28
Output: "AB"

Example 3:

Input: columnNumber = 701
Output: "ZY"

不愧是easy难度的题目，就是给人一种我上我行的感觉。这个题目本质上就是实现一个进制转换，满26进1嘛。 先写一个数字转字母的函数

利用ascII，将1到26的数字转变为字母，这里注意的是，如果是0，则返回Z。 ### 解法

我这里定义了head和tail。一个是向下取整除，一个是余数。
